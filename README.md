# Wordpress training website.

Wordpress training website built on Lando.

## Dependencies
* Docker
* Lando
* Composer

## Installation

```bash
lando install
```

## Usage

```bash
lando start
```

### Import database
```bash
lando db-import
```

### Export database
```bash
lando db-export
```
